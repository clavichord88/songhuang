Elastic Load Balancer

4 kinds of load balancers offered by AWS

  • Application Load Balancer(HTTP/HTTPS only) - Layer 7
  • Network Load Balancer (ultra-high performance, allows for TCP) - Layer 4
  • Gateway Load Balancer - Layer 3
  • Classic Load Balancer (retired in 2023) - Layer 4 & 7

Components for Load Balancer

  • Multiple EC2 instances, which are responsible for handling the requests from users
  • Then, we need to a load balancer and put the load balancer into a security group
  • Inside the security group, we need to have a target group to group the EC2 instances together, then the load balancer could point to the target group
